# Rick and Morty

> Rick and Morty Characters Web Page.

## Getting Started

To get a local copy up and running follow these simple example steps.

- Clone the project:

```
git clone https://github.com/luftedar/rickandmorty.git
```

- cd into the folder

```
cd rickandmorty
```

- Install NPM

```
npm install
```

- Open with a code editor

```
code .
```

- Start local server

```
npm start
```

- For testing

```
npm run test
```

### Prerequisites

- Text Editor
- Web Browser
- NPM
